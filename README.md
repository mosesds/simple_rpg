# simple_rpg

My first side project

-A simple RPG: 
	-Renders map
	-Impliments HUD
		-HP, EXP, LVL, Buffs
	-Character/Mob animations
	-Combat
		-Shoot a projectile
		-Mobs do touch damage
		-Display each mobs HP
		-Respawn mob upon its death
		-Knockback/iFrames when hit
	-EXP/Leveling
		-Character stats increase upon level up
